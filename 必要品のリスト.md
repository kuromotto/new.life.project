# 新生活スタートのための準備

- 参考資料
    - [【大学から一人暮らしを始める方へ】新生活で必要なものは？ | ヤマダデンキ](https://www.yamada-denkiweb.com/media/1322/)

## 初日から必要_手持ち持参
- [ ] マイナンバーカード
- [ ] 印鑑
- [ ] PC どちらか1台
- [ ] スマートフォン、スマートフォン充電器、モバイルバッテリー

## 初日から必要（優先度S+）

- [ ] 抱き枕
- [ ] 布団
- [ ] 歯磨き歯ブラシ
- [ ] トイレットペーパー
- [ ] ディッシュペーパー
- [ ] ペーパータオル
- [ ] ハンドソープ
- [ ] バスタオル
- [ ] 普通のタオル
- [ ] 紙皿、紙コップ、割り箸
    - これがあると当面は皿洗いしなくてもやり過ごせる
- [ ] 食器用洗剤
- [ ] スポンジ
- [ ] 洗いカゴ
- [ ] 包丁、まな板
- [ ] ヘアドライヤー
- [ ] 着替え数日分
- [ ] お風呂用品（石鹸またはボディソープ・シャンプー）
- [ ] 風邪薬
- [ ] 胃薬
- [ ] 絆創膏
- [ ] 体温計（既設流用）
- [ ] 耳かき

## 1〜2日以内に必要（優先度S）

- [ ] 乾電池（リモコン用）
- [ ] テーブルタップ、コンセントキャップ
- [ ] ゴミ箱
- [ ] PC 他方の1台
- [ ] 衣装ケース
- [ ] 洗濯用洗剤
    - 室内干しに対応した洗剤がよい
- [ ] 室内干し用のスタンドみたいなやつ
- [ ] ハンガー
- [ ] 入学式用スーツ
    - 事前に着合わせしておく必要あり
- [ ] 風呂の排水溝用のフィルターみたいなやつ


## 数日以内に必要（優先度A）

- [ ] 冷蔵庫
- [ ] 洗濯機
- [ ] 炊飯器
- [ ] 電子レンジ
- [ ] 電気ストーブ
- [ ] 電気ケトル（t-falみたいなやつ）
- [ ] 自転車
- [ ] 調味料
- [x] IH対応調理器具 （2025/12/30 IH対応フライパン（蓋・ターナー・レードル付き）を購入@ヒラキ）
- [x] 入学式用の靴 （2025/12/30 黒い靴を購入@ヒラキ）
- [ ] 座布団

## 優先度を検討し購入（優先度B）

- [ ] カトラリー一式（スプーン、フォークなど）
- [ ] テレビ
- [ ] BD/DVDプレイヤー
- [x] 絨毯・カーペット （2025/12/30 電気毛布を購入@ヒラキ）
- [x] こたつ （2025/12/30 電気毛布を購入したのでこたつは当面不要
- [ ] レンジラック
    -  [ニトリ レンジラック](https://www.nitori-net.jp/ec/product/2110200009319s/?srsltid=AfmBOorL0P2UynWlljrDfldFPOPAu0s5Fd3MgijAhyCPAwcPcNo9izBz)
- [ ] サーキュレーター：洗濯物の室内干しの乾燥用
- [ ] 机と椅子

## 要否検討

- [ ] アイロン
    - 形状記憶のワイシャツを選んでおけば、ワイシャツのシワ取り用としてはなくても良い。洗濯物が乾ききらないときに便利なときはある。
- [ ] プリンター
    - 学期開始後に必要度を見極めてからで十分。

# 前広に準備が必要

## 1月中

- [ ] 引っ越し手配
- [ ] 住民票転出届 ・・・ [マイナポータルで手続き可能](届出・手続き.md)
- [ ] 銀行口座
- [ ] 印鑑
- [ ] 仮置き用のダンボール箱
    - [ ] キャンパスフェスティバル時に使ったダンボール転用でよい

## 3月入ってからでOK

- [ ] [インターネットプロバイダー](インターネット.md)
    - 転居先マンションに来ているのは光ファイバー回線
- [ ] 郵便局の転居届：[e転居](https://lp.jpetn.pf.japanpost.jp/guide/)


## 引っ越してから

- [ ] 住民票転入届、マイナンバーカード住所変更届け
